# Students Performance – Exploratory Data Analysis

This project applies Exploratory Data Analysis (EDA) techniques to the Students Performance dataset.

## Dataset
- Source: Kaggle – Students Performance in Exams
- File: StudentsPerformance.csv

## Contents
- EDA_Students_Performance.ipynb: Jupyter Notebook containing the full EDA process
- StudentsPerformance.csv: Dataset used for analysis

## EDA Steps
- Data loading and inspection
- Statistical summary
- Missing values analysis
- Data visualization
- Insights and conclusions
